### 目录结构
目前主要存放四个模块代码：
- Base：基类，桥接文件
- Extension：项目中使用到的分类
- UIComponents：封装UI组件库，常用的父类等
- VenderLib：第三方库，如：加密解密签名验签的库
- Tools：工具库，如：数据缓存处理、文件操作等


### 使用

```Ruby
pod 'QNNLib'
```
